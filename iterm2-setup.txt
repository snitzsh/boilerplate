# Terminal (not vim) Set Up
https://www.freecodecamp.org/news/jazz-up-your-bash-terminal-a-step-by-step-guide-with-pictures-80267554cb22/

In terminal do
```sh
echo "source $(brew --prefix)/opt/powerlevel10k/powerlevel10k.zsh-theme" >>~/.zshrc
```

Vim Airline installation
```sh
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

```
Once Vim Plug is installed, edit ~/.vimrc:
```
call plug#begin('~/.vim/plugged')
Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'
call plug#end()
```
# Close Vim and open Vim again (without anyfile)
and type:
```
:PlugInstall
```

# Clone Vim Airline Fonts
```
$ git clone https://github.com/powerline/fonts.git --depth=1
$ cd fonts
$ ./install.sh
$ cd ..
$ rm -rf fonts
```

Set up fzf
Add this line in ~/.zshrc
```
[ -f ~/.fzf.zsh ] && source ~/.fzf.zsh

Add this line in ~/.vimrc
```
set rtp+=/opt/homebrew/bin/fzf
```

```sh
fzf usage inside vim, type
```sh
:FZF
```

Install vim theme
```
$ git clone https://github.com/joshdick/onedark.vim.git ~/onedark
$ cd ~/onedark
$ mkdir ~/.vim/colors
$ cp colors/onedark.vim ~/.vim/colors/
$ cp autoload/onedark.vim ~/.vim/autoload/
```
